# WIK News Portal 📰

A dynamic, responsive, and fully structured news website built to deliver the latest global updates, sports news, and trending stories. 

## 🚀 Overview
WIK News is a frontend web application designed with a clean and minimalistic user interface. It categorizes various news streams—including politics, global events, and sports—into dedicated sections, providing users with a smooth and intuitive browsing experience.

## ✨ Features
*   **Categorized News Pages:** Dedicated pages for different news topics (e.g., Global Politics, Sports, Live Updates, Videos).
*   **Structured Routing:** Organized folder architecture separating core pages, sub-news pages, and assets for clean code maintainability.
*   **Responsive Design:** Styled with custom CSS to ensure readability across different screen sizes.
*   **Dynamic Elements:** Interactive UI components powered by Vanilla JavaScript.

## 🛠️ Tech Stack
*   **HTML5:** Semantic structure and multi-page routing.
*   **CSS3:** Custom styling, layouts, and responsive design (`style.css`, `style_live.css`, etc.).
*   **JavaScript:** Vanilla JS for frontend logic and interactivity (`script.js`).
